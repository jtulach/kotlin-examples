Install GraalVM

To use this project you need GraalVM Java virtual machine. Download it from here. Then invoke following commands to convert the sample application into native one and test it: 

$ JAVA_HOME=/path/to/graalvm mvn install -Psubstratevm
$ ./hello.hellokt

Compare the startup time of regular Java application and native one generated by GraalVM's native-image tool: 

$ time ./hello.hellokt 
Hello, world!

real    0m0.001s
user    0m0.000s
sys     0m0.000s

GraalVM can convert your kotlin code and all Java libraries it is using into binary that starts instantly!
